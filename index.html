In this blog post, I will share my Cloud School Experience: Learning with She Code Africa Cloud School Program and highlight the key aspects of the curriculum and training schedule that made this program a transformative experience.

Cloud computing has become an integral part of the technology industry, revolutionizing the way businesses operate and store data. As a passionate learner in the field of technology, I recently had the opportunity to participate in the She Code Africa Cloud School Program, an immersive learning experience designed to equip individuals with the necessary skills and knowledge in cloud computing. Here are a list of blogs showcasing what I learnt through the program.


Cloud Computing - Level 1

The program began with an introduction to cloud computing, providing an overview of the different cloud computing models and components. We explored cloud solutions and services, as well as emergent trends and practices in the industry. This foundational knowledge set the stage for deeper exploration of cloud platforms.

Wendy Nina

Apr 9, 2023 3 min

AWS Fundamentals and Google Cloud Fundamentals

During the first weekend, we delved into the fundamentals of two prominent cloud platforms: Amazon Web Services (AWS) and Google Cloud. We learned how to set up accounts, gained an overview of their services, and explored key features such as EC2, S3, VPCs, IAM, and security best practices. The program encouraged hands-on learning, and we were tasked with creating a simple infrastructure using EC2 and S3 in AWS, as well as Compute Engine and Cloud Storage in Google Cloud.

Wendy Nina

Apr 16, 2023 2 min

Core Azure Concepts and Services

Moving forward, the curriculum shifted its focus to Microsoft Azure, one of the leading cloud computing platforms. We dived into Azure's fundamental concepts and architectural components, gaining insights into Azure compute services, storage services, analytics, and databases. The program emphasized practical skills, and we were challenged to create an Azure App Service to host a web application.

Wendy Nina

Apr 23, 2023 4 min

Azure Security, Identity, and Governance

As we progressed, the program covered crucial aspects of cloud security and network security features in Azure. We learned how to protect against threats, secure network connectivity, and implement identity services. Additionally, we explored privacy, compliance, and data protection standards on Azure. These lessons provided a comprehensive understanding of how to ensure the security and compliance of cloud-based applications and services.

Wendy Nina

Apr 30, 2023 4 min

Azure SQL and Database Management

Another vital topic covered in the program was Azure SQL and database management. We learned how to deploy and configure servers, instances, and databases in Azure SQL, exploring different database options. Backup and restore procedures were also covered, along with securing data in Azure SQL and achieving consistent performance and high availability.

Wendy Nina

May 6, 2023 3 min

Azure Applications and Microservices

In the later stages of the program, we delved into deploying applications to Azure, understanding the benefits and costs of cloud deployments. We explored different service platforms and gained insights into connecting cloud storage to cloud applications. Furthermore, we delved into microservices, Docker, and Kubernetes, implementing serverless microservice architectures using Azure cloud serverless offerings.

Wendy Nina

May 7, 2023 3 min

Azure Migration and Performance

The program concluded with a focus on Azure migration and performance. We explored different aspects of cloud migration, including migrating web applications, database workloads, and background job processing services. Additionally, we learned about monitoring application performance using Azure's Application Insights and management tools.

Wendy Nina

May 14, 2023 7 min

Project Week

In the final week, we were given the opportunity to apply our newly acquired knowledge and skills to a hands-on project. This project allowed us to demonstrate our understanding of cloud computing concepts and showcase our ability to deploy applications and manage cloud resources effectively.
